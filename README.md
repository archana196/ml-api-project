# 🔗 Machine Learning Model Deployment using API

## 📌 Problem Statement
Machine learning models are useful only when deployed for real-time usage.  
This project focuses on serving ML predictions through an API.

---

## 🎯 Objective
- Deploy a trained ML model  
- Provide real-time predictions using API  
- Simulate real-world ML deployment  

---

## ⚙️ Implementation

- Trained a machine learning model  
- Built API using Flask  
- Created endpoint for predictions  

---

## 🚀 API Usage

### Endpoint:
/predict

### Input:
{
  "feature1": value,
  "feature2": value
}

### Output:
{
  "prediction": result
}
